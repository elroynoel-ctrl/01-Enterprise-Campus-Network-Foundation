# Enterprise Campus Network Foundation

## Full Stack Network Associate (FSNA) Program


This project documents the foundational enterprise network lab completed as part of my hands-on Cisco networking portfolio.

The objective of this lab was to build a functional enterprise network using Cisco routers and switches while applying industry best practices for device configuration, management, and verification.

---

# Objectives

- Configure Cisco routers and switches
- Configure management VLAN
- Secure device access
- Configure console and VTY access
- Configure local authentication
- Verify Layer 2 connectivity
- Verify Layer 3 connectivity
- Document all configurations

This lab established the foundation for all future enterprise networking projects. Every advanced routing technology—including OSPF, EIGRP, and BGP—depends on a properly configured Layer 2 and Layer 3 infrastructure.
